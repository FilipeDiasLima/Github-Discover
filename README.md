# How to run
```
 # Clone repository
 $ git clone https://github.com/FilipeDiasLima/Github-Profiles.git
```
 
 ### Install libs
```
 # cd Github-Profiles/web
 $ yarn
```
 ### Run web project
 ```
 # cd Github-Profiles/web
 $ yarn start
```
